A Tesla website clone built using **HTML & CSS**.  
This project replicates the sleek design of Tesla’s official site, created as a practice project to strengthen front-end development skills.

---

Preview
Live Demo: [Tesla Clone](https://srujan-s33.github.io/tesla-clone/)

---

Tech Stack
- HTML5 – Semantic structure  
- CSS3 – Styling with Flexbox & Grid  
- Responsive Design – Works on all devices  

---

Features
- Tesla-inspired UI & layout  
- Smooth scrolling between sections  
- Fully responsive design (desktop & mobile)  
- Minimal and modern styling
